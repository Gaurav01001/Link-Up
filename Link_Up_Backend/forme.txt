Client sends: POST /auth/register
  ↓
Express parses JSON body (express.json middleware)
  ↓
Matches route → /auth/register → register controller
  ↓
Controller validates with Zod
  ↓
Calls registerUser service
  ↓
Service checks DB for duplicates
  ↓
Hashes password with bcrypt
  ↓
Saves user to PostgreSQL via Prisma
  ↓
Creates JWT with user ID
  ↓
Returns to controller
  ↓
Controller sends 201 response with token



Client sends: GET /auth/me with Bearer token
  ↓
Express matches /auth/me
  ↓
authenticate middleware runs FIRST
  ↓
Extracts token from header
  ↓
Verifies JWT (is it valid? expired?)
  ↓
Finds user in DB by decoded userId
  ↓
Attaches user to req.user
  ↓
Calls next()
  ↓
getMe controller runs
  ↓
Sends req.user back as response

Code	Meaning	When
